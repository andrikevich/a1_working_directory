Как прошивать через fastboot

Сохранить папку fastboot в корень вашего диска [C:\]
Извлечь из архива Kernel.elf / boot.img
Сохранить Kernel.elf / boot.img в папку с fastboot (возле файла fastboot.exe)
Выключить устройство
Загрузить устройство в режиме fastboot (Громкость ВВЕРХ + USB кабель подключается к компьютеру -> FASTBOOT MODE)
Открыть командную строку (cmd)

///////////////////////////////
   {Смена диска в cmd 
   В случае если необходимо поменять не только текущий каталог, но и текущий диск, необходимо использовать ключ /D. Команда будет выглядеть так: cd /D [диск:/].
Например, перейдем из папки «Program Files», расположенной на диске «C», на диск «D»: cd /D d:/} 
//////////////////////////////////////
Обязательно использовать команду ниже, что-бы перейти в каталог с фастбутом.
cd c:\fastboot

- Эта команда для проверки что телефон подключен верно и драйвера установлены. 
Код

fastboot.exe -i 0x0fce getvar version

Если ответ был например 0.3, то это означает что телефон подключен верно. 
Если выдаёт waiting for device, проверяем состояние USB подключения. Если присутствует неизвестное устройство в диспетчере устройств, то проверяем установку драйвера.

использовать следующие команды, чтобы прошить ядро kernel.elf
fastboot flash boot kernel.elf

Так-же можно использовать эту команду чтобы прошить ядро в формате .img
fastboot flash boot boot.img

или эту команду чтобы прощить систему в формате .img
fastboot flash system system.img

После прошивки загрузиться в рекавери дабы проверить что ядро прошилось.
Перезагрузить телефон и пользоваться.

Примечание:
Файл ядра kernel.elf можно переименовать как вам удобно, к примеру XSXKernel_v6.elf или DooM.elf или KASpeed.elf и положить их все рядом. При этом в командной строке вы должны указать нужное вам имя ядра.

Как прошить ядро?

1. С помощью программы QuickIMG
2. C помощью программы FlashTool
3. C помощью программы Fastboot: .стрела. 
- Скачиваем архив с fastboot casete.ru/21695 и распаковываем его например на диск С:
- Откройте командную строку Windows (win+r - cmd) и перейдите в папку с fastboot
Код
с:
сd \fastboot
- Подключаем телефон к компьютеру с помощью USB шнурка в режиме fastboot (в выключенном телефоне зажать кнопку меню (правая) и вставить кабель USB в телефон) - должен загореться синим цветом светодиод возле кнопки включения телефона. (Естественно другой конец USB уже должен быть включен в компьютер)
Примечание: Если Windows обнаружил новое устройство и требует драйвера, укажите ему паку куда вы распаковали архив Fastboot - в ней есть необходимые драйвера.
- Скопируйте нужное ядро для прошивки в папку с fastboot (c:\fastboot) (в дальнейшем подразумевается, что ядро имеет имя boot.img)
- Эта команда для проверки что телефон подключен верно и драйвера установлены. 
Код
fastboot.exe -i 0x0fce getvar version
Если ответ был например 0.3, то это означает что телефон подключен верно. 
Если выдаёт waiting for device, проверяем состояние USB подключения. Если присутствует неизвестное устройство в диспетчере устройств, то проверяем установку драйвера.
- Если все в порядке, прошиваем ядро: (команда прописана с учетом, что файл ядра называется boot.img)
Код
fastboot -i 0xfce flash boot boot.img
- Если возникла ошибка что файл не может быть прошит - проверьте что ядро лежит в той же папку где и fastboot и называется boot.img
- Если выходит ошибка что ядро не может быть прошито так как необходим рут, значит bootloader у вас НЕ разблокирован.
