Branch View Extended 1.03
Branch View Extended показывает все вложенные файлы и папки, расположенные по заданному пути, одним списком. Размеры папок подсчитываются.


ИСПОЛЬЗОВАНИЕ.

Создайте кнопку или пункт меню со следующими параметрами:

Команда: cd
Параметры: \\\BranchViewEx\%p

Плагин работает для текущего пути, как и стандартный Branch View.


ПРИМЕР ИСПОЛЬЗОВАНИЯ.

Плагин, среди прочего, может пригодиться, если нужно быстро освободить место на диске. В этом случае можно поставить метод сортировки "по размеру" и запустить плагин. Тогда плагин поможет найти не только самые большие из ненужных файлов (с чем отлично справляется и стандартный Branch View), но и большие папки, содержащие множество небольших файлов (поэтому их не найти с помощью стандартного Branch View) и похороненные глубоко в файловой системе (поэтому их трудно найти вручную).


Плагин делался для себя, поэтому не особо тестировался и довольно кривоват по части интерфейса.

"Особенности":
-- Нельзя открывать папки внутри плагина.
-- Плагин не запустится из Сетевого окружения.
-- Папки показаны как файлы с расширением "./" (Поэтому, если вы назначили цветовую схему для папок, рекомендую назначить ее и для файлов "*./").
-- Плагин не заходит по символическим ссылкам во время сканирования, но показывает их в начале списка как папки (ссылки, кстати, можно открывать внутри плагина).
-- Для отображения процесса сканирования приспособлен стандартный диалог копирования.
