 This looks like a configuration issue of the map layers.

1.	Close Actix software, if running.
2.	Open this location C:\ProgramData\Actix\Analyzer\Bin.
3.	Delete AppSettings.dat
4.	Go to Programs and Features and repair Actix software.

1. Copy the following location %ALLUSERSPROFILE%\Application Data\Actix\Analyzer\Bin

3. Delete any activation.xml or lservrc files 
4. Copy the following location %ALLUSERSPROFILE%\Application Data\Actix\Analyzer\ 
5. Paste it into a Windows Explorer Window. 
6. Delete any activation.xml or lservrc files
7. From the Windows Start menu, select Programs, Actix, Analyzer – Activation Wizard
8. Ensure the top option is selected and click Next. 
9. Enter your Product IDs above and click Next.
10. Click Done.
